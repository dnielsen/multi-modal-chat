This open source chatbot demonstrates Q&A using AI with the multiple layers of sophistication, such as:
- One question via text with one reply from an LLM only, from terminal window
- One question via text with one reply from an LLM only, from web interface 
- One question via text with one reply from an LLM and vector database (RAG), from web interface
- One question via both text & image with one reply from an LLM, from web interface 
- Multiple questions with multiple replies (using A2A) from LLM
