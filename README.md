This open source chatbot demonstrates the following levels of sophistication, such as:
- One Question via text with one Answer from LLM only, from terminal window
- One Question via text with one Answer from LLM only, from web interface 
- One Question via text with one Answer from LLM and vector database (RAG)
- One Question via text & image with one Answer from LLM and vector database (RAG)
- Multiple Questions with multiple Answers from LLM and vector database (RAG) using A2A
